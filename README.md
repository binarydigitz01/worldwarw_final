# World War W

[Here is a sample of our game!](./screenshots/img1.png)

This was a game built during the smart india hackathon 2024, with the Main Theme being "Ground Water Conservation".
With water as the currency, you buy equipment to farm water, but if you farm too much, ground water will run out!

This is the project submission for Smart India Hackathon 2024 by Arnav Vijaywargiya, Siya Rath, Aditya Niggam, Mrityunjay Bhatt, Urja Srivastava and Asna.
